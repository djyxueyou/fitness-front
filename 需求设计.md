# 《铁记》V1.0 产品需求文档

## 一、产品概述

### 1.1 产品定位

健身房自主训练者的 **“训练日记 + 动作字典”** 工具。用最轻量的交互记录每一次训练，用清晰的动作演示辅助每一次动作。

### 1.2 目标用户

- 在健身房自主训练的力量训练爱好者
- 需要记录训练数据、追踪进步的训练者
- 需要查阅动作要领和演示的健身新手

### 1.3 V1.0 核心价值

- **练前**：浏览动作库，学习动作要点，收藏常用动作
- **练中**：极速记录每组数据，自由切换动作
- **练后**：查看训练历史，添加笔记，复用为模板

### 1.4 产品形态

微信小程序（uni-app 开发）

## 二、全局说明

### 2.1 用户身份

所有用户登录后即可使用全部功能，**V1.0 无会员体系**，无功能限制。

### 2.2 数据同步

用户训练数据自动同步至云端（微信云开发数据库），换设备可恢复。

### 2.3 单位系统

- 默认重量单位：**kg**
- 支持在“个人设置”中切换为 **lb**
- 切换后，历史数据展示时自动换算（存储时始终以 kg 为单位）

### 2.4 底部导航栏

| Tab  | 名称   | 功能                                     |
| :--- | :----- | :--------------------------------------- |
| 1    | 训练   | 开始训练入口、今日训练快捷入口、最近模板 |
| 2    | 动作库 | 动作分类浏览、搜索、收藏                 |
| 3    | 我的   | 个人中心、历史记录、模板管理、设置       |

## 三、模块一：训练记录（核心）

### 3.1 开始训练流程

text

```
首页点击“开始训练” 
    → 进入“选择模板”页
        → 可选择已有模板
        → 或点击“空白训练”（直接开始，无预设动作）
    → 进入训练中页面
```



**选择模板页**：

- 展示用户创建的所有模板卡片（按最近使用排序）
- 每个卡片显示模板名、包含动作数
- 底部固定“空白训练”按钮

### 3.2 训练中页面（核心交互）

#### 3.2.1 页面结构

text

```
┌─────────────────────────────────┐
│  ← 返回    训练时长 00:25:30    │
├─────────────────────────────────┤
│                                 │
│     [当前动作 GIF 循环播放]      │
│                                 │
│        平板杠铃卧推              │
│        第 3 组 / 共 4 组         │
│                                 │
│  ┌─────────┐    ┌─────────┐    │
│  │  60.0   │    │   8     │    │
│  │   kg    │    │   reps  │    │
│  └─────────┘    └─────────┘    │
│                                 │
│      [  完成一组  ]              │
│                                 │
├─────────────────────────────────┤
│ 动作切换栏（横向滑动）            │
│ [卧推] [飞鸟] [夹胸] [推举] ...  │
├─────────────────────────────────┤
│ 组间休息 01:28    [跳过]         │
└─────────────────────────────────┘
```



#### 3.2.2 交互说明

| 元素             | 交互                           | 说明                         |
| :--------------- | :----------------------------- | :--------------------------- |
| **重量输入**     | 点击数字区域弹出数字键盘       | 支持小数点后一位（如 62.5）  |
| **次数输入**     | 点击数字区域弹出纯数字键盘     | 整数                         |
| **完成一组**     | 点击后记录当前重量×次数        | 自动进入组间休息倒计时       |
| **组间计时器**   | 完成一组后自动启动，可手动跳过 | 默认 90 秒，可在设置中调整   |
| **动作切换栏**   | 横向滑动，点击切换动作         | 展示今日训练的全部动作       |
| **GIF 区域**     | 点击可暂停/播放                | 加载时显示占位图             |
| **当前动作卡片** | 点击 GIF 区域或动作名          | 跳转该动作详情页（查看要领） |

#### 3.2.3 核心机制：顺序引导 + 自由切换

- **默认模式（顺序引导）**：
  - 用户从模板开始时，系统按模板预设顺序排列动作
  - 完成一个动作的所有组后，**自动切换到下一个动作**
  - 切换时有轻微动效提示
- **自由切换模式**：
  - 用户可随时通过底部动作栏切换到任意动作进行记录
  - 不强制完成当前动作所有组才能切换
  - 训练结束时，未完成所有组的动作正常保存

#### 3.2.4 添加/删除动作

- 训练过程中，用户可点击动作栏末尾的 **“+”** 按钮
- 弹出动作库选择器，可搜索/浏览并添加新动作到本次训练
- 长按动作栏的动作标签，可删除该动作（及其已记录的所有组）

### 3.3 完成训练

- 点击右上角“完成”按钮

- 弹出确认弹窗：

  text

  ```
  ┌─────────────────────────┐
  │   本次训练摘要            │
  │   时长：62 分钟           │
  │   总组数：24 组           │
  │   总容量：6,240 kg        │
  │                          │
  │   [添加笔记]              │
  │   ┌──────────────────┐   │
  │   │ 今天状态不错...    │   │
  │   └──────────────────┘   │
  │                          │
  │   [取消]    [保存训练]    │
  └─────────────────────────┘
  ```

  

- 笔记为可选字段，字数上限 200 字

- 保存后跳转至“训练完成页”

### 3.4 训练完成页

- 展示本次训练完整数据（每个动作×每组×重量×次数）
- 底部按钮：
  - **分享**：生成训练总结卡片（图片），可保存或分享
  - **另存为模板**：将本次训练的动作列表保存为新模板
  - **完成**：返回首页

### 3.5 历史记录

#### 3.5.1 列表页

- 按日期倒序排列，每月一个分组标题
- 每条显示：日期、训练时长、总容量、动作数量
- 左滑可删除

#### 3.5.2 详情页

- 展示该次训练的完整数据（动作×组数×重量×次数）
- 右上角“编辑”按钮

#### 3.5.3 编辑历史记录

- **允许修改任意历史记录**（包括 3 天前）
- 点击“编辑”进入编辑模式：
  - 可修改每组重量/次数
  - 可删除某组
  - 可添加新组
  - 可添加新动作（从动作库选取）
- 保存后直接覆盖原记录，**不标记“已编辑”**

## 四、模块二：动作库

### 4.1 动作分类

V1.0 基于你提供的 8 大分类：

| 分类 ID   | 分类名 | 说明                   |
| :-------- | :----- | :--------------------- |
| chest     | 胸部   | 卧推、飞鸟、夹胸等     |
| back      | 背部   | 划船、引体、下拉等     |
| legs      | 大腿   | 深蹲、腿举、腿屈伸等   |
| shoulders | 肩膀   | 推举、侧平举、前平举等 |
| arms      | 手臂   | 弯举、下压、臂屈伸等   |
| core      | 核心   | 卷腹、举腿、平板支撑等 |
| glutes    | 臀部   | 臀推、髋外展等         |
| calves    | 小腿   | 提踵等                 |

### 4.2 动作列表页

- 顶部 Tab 栏：全部 / 胸部 / 背部 / 大腿 / 肩膀 / 手臂 / 核心 / 臀部 / 小腿
- 支持搜索（按动作名）
- 每个动作卡片显示：GIF 缩略图（首帧）、动作名、目标肌群标签
- 右上角收藏图标（实心/空心）
- 默认按“使用频次”排序（被记录次数越多越靠前）

## 五、模块三：训练模板

### 5.1 模板管理页

- 入口：“我的” → “模板管理”
- 列表展示用户创建的所有模板
- 支持新建、编辑、删除

### 5.2 创建/编辑模板

text

```
┌─────────────────────────────────┐
│  取消        新建模板        保存 │
├─────────────────────────────────┤
│  模板名称：[  推日  ]            │
│                                 │
│  动作列表：                      │
│  ┌─────────────────────────┐   │
│  │ 平板杠铃卧推       4组   │   │
```

## 六、模块四：数据看板

### 6.1 入口

- “我的”页面顶部“训练日历”卡片，点击进入

### 6.2 训练日历

- 月历视图，左右滑动切换月份
- 有训练记录的日期显示**绿色圆点**
- 点击日期，底部弹出当日训练摘要：
  - 训练部位/模板名
  - 训练时长
  - 总容量
  - “查看详情”按钮跳转历史详情

### 6.3 容量趋势图

- 展示在训练日历下方
- 默认展示“本周容量趋势”（柱状图）
- 横轴：周一至周日
- 纵轴：训练总容量（kg）
- 支持切换“上周”对比

## 八、数据字段定义

### 8.1 训练记录（TrainingRecord）

| 字段            | 类型     | 说明                    |
| :-------------- | :------- | :---------------------- |
| id              | string   | 唯一标识                |
| userId          | string   | 用户 ID                 |
| templateId      | string   | 使用的模板 ID（可选）   |
| startTime       | datetime | 开始时间                |

## 十二、需求确认清单

请确认以下关键点：

- 训练中页面交互：顺序引导 + 自由切换，符合预期
- 历史记录编辑：允许修改任意记录，不标记已编辑
- 单位切换：默认 kg，支持切换 lb，历史数据自动换算
- GIF 分类：8 大分类，名称已确认
- 动作详情页：V1.0 文字说明留空，仅展示 GIF 和基础信息
- 另存为模板：从历史记录一键保存
- 训练笔记：支持，最多 200 字
- 无会员体系：所有功能免费开放
- 热身组：不做

确认无误后，可进入 UI 设计和开发排期阶段。
